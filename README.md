# vue2-project

A simple app to write notes. The text in the textarea is compressed and added the the url as a parameter so that the link with the url data parameter can be sent to someone else or loaded into another browser.

## Usage
1) Type in the textarea
a) If there is no text in the textarea, start by typing in the textarea.
b) If there is already text in the textarea, simply add your own text and the url will update automatically.

2) Once you're finished your note, copy the entire url, including the ?data=[value] and share it, or paste it somewhere like a new browser.
